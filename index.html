JazzyFS Sonification Audio

These audio files are generated from the macOS/APFS JazzyFS decision logs. The first table contains post-workload music summaries. The second table contains real-time seek-tone audio generated from adaptive-mode seek deltas.

For post-workload summaries, the root note can come from the natural-note set A through G. The prefetching mode controls the scale quality.
